내지 신청 시 주의사항

✔︎ 디자이너가 작업할 페이지 수를 주문서에 같이 기입해주세요. (회지 전체 페이지수 X)
(예시) 목차 1p, 장표지 4p로 총 5p입니다. 장표지는 모두 우측면으로 들어갈 예정이고 A타입 내지를 원합니다.
✔︎ 차례의 경우 2p(좌우양면)로 진행되는지 1p(우측면)로 진행되는지 꼭 알려주세요.
✔︎ 장표지의 경우, 제목이 추상적이거나 짧을 경우 키워드나 줄거리를 반드시 포함해주세요.
✔︎ 보통은 장소나 키워드 중심으로 작업하기 때문에, 소설 내의 장소를 언급해주셔도 좋습니다.
✔︎ 내지는 pdf 낱개로 제공됩니다. (zip 압축파일로 제공하지 않음)

Type A ; 사진 타입

배경에 사진 또는 간단한 그라데이션이 들어갑니다.
흑백 소설 장표지로 진행할 경우 매우 효과적입니다.

Type B ; 일러스트 타입

간단한 일러스트, 라인, 패턴이 들어가는 내지 타입입니다.
사진보다 직설적이고 강렬한 인상을 남기기 좋습니다.
너무 복잡한 일러스트 요청은 불가합니다.

차례 샘플

차례(1p) 샘플 이미지의 좌측면 색지는 임의로 넣은 것이며, 실제로는 제공되지 않습니다.
